## 📘 Resumo
Este projeto tem como finalidade o desenvolvimento de um sistema fundamentado em banco de dados relacional e controle de versão. A proposta busca oferecer uma solução tecnológica capaz de organizar e manipular informações relacionadas à segurança ocupacional.

**Palavras-chave:** Banco de Dados, Segurança do Trabalho, Controle de Versão, Git.
